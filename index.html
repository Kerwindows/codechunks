PHP: REMOVE EVERYTHING IN A STRING THAT COMES BEFORE A CHARACTER, INCLUDING THE CHARACTER

Code Description

This can be achieved using string manipulation functions in PHP. First we find the position of the desired character in the string using strpos(), substr(), ltrim, and trim()

            
        $new_name = substr($old_name, strpos($old_name, '_') + 1);

or

            
        $new_name = ltrim(stristr($old_name, '_'), '_');

How this code works?

The above code was used to fix a problem with the global variable $_GET[] when calling pages on the index.php file for back-end applications. The code was used to check for string after the first question sign (?) and then usaed again to find the first equal sign after the current page URL using $_SERVER["REQUEST_URI"]. This process requires two stages.

Part 1: Remove question mark and everything before it 

            
//get ur
//global variable:

$question_mark_check = ltrim(stristr($_SERVER["REQUEST_URI"], '?'), '?')
//output: 
studentprofile=C3836B2C

//set input variabl

Issues implementing this code

There was one issue encountered when using this code. The code returns an empty value iF the url does not have an equal sign in the string. To resolve this an if statement was used.

            
function getSlug()
{
  //check if url has the = sign, if no then trim everything before the '?' including the '?'
  if (empty(substr($_SERVER["REQUEST_URI"], 0, strpos($_SERVER["REQUEST_URI"], "=")))) {
    return ltrim(stristr($_SERVER["REQUEST_URI"], '?'), '?');
  }
  //else if the = sign is present then trim the = sign and everything after it also
  else {
    $question_mark_check = ltrim(stristr($_SERVER["REQUEST_URI"], '?'), '?');
    return substr($question_mark_check, 0, strpos($question_mark_check, "="));
  }
}

How this code was used?

        include (CUC_VIEW_PATH . '/studentprofile.php');
    break;
    default:
        include (CUC_VIEW_PATH . '/dashboard.php');
}

PHP:HOW TO CALCULATE AGE?

Code Description

The age of an individual can be calculated from the user's date of birth. the user's date of birth must be in the format yyyy-mm-dd.

        

$birth_date = strtotime('Y-m-d');
$now = time();
$age = $now-$birth_date;
$a = $age/60/60/24/365.25;
echo floor($a);

How this code was used?
